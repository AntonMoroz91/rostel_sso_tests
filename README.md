## 🚀 Установка и запуск
### 1. Клонирование
git clone https://github.com/AntonMoroz91/rostel_sso_tests.git
cd rostel_sso_tests

### 2. Окружение
python -m venv .venv
source .venv/bin/activate (Linux/Mac)
.venv\Scripts\activate (Windows)
pip install -r requirements.txt

### 3. Запуск
pytest -v
pytest tests/test_auth.py::TestAuth::test_auth_page_loaded -v -s

⚠️ Требуется Chrome 147+, VPN с российским IP (если за рубежом)

## 💡 Обоснование автоматизации
🔧 Автоматизированы: сценарии без SMS/email, стабильные локаторы, критичный функционал
✋ Ручные: коды из SMS/email, таймер, всплывающие окна
💬 Причина: требуется доступ к SMS-шлюзам, почтовым API или мокам
